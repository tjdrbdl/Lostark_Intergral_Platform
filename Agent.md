# Agent.md

## 프로젝트명
로스트아크 데이터/지표 통합 사이트

## 역할
- ChatGPT: 기획자
- Codex: PM
- VSCode Chat: 개발자

## 핵심 원칙
1. 폴더 구조보다 먼저 유저 시나리오와 데이터 흐름을 확정한다.
2. 프론트는 외부 데이터 소스를 직접 알지 않는다.
3. 백엔드는 외부 응답을 내부 표준 스키마로 정규화한다.
4. 캐시는 기본 전제로 설계한다.
5. 실패 시 전체 실패보다 부분 성공 응답을 우선한다.
6. MVP 범위를 벗어나는 기능은 backlog로 분리한다.

## MVP 핵심 페이지
1. 홈 / 검색 랜딩
2. 캐릭터 / 원정대 통합 결과 페이지
3. 원정대 / 주간 체크 보조 페이지
