# ArkPilot MVP PRD

최종 갱신: 2026-07-13

## 제품 한 줄 정의

캐릭터명을 검색하면 원정대 전체의 성장 상태를 분석해, 이번 주 어떤 캐릭터에 무엇을 먼저 투자할지 근거와 함께 보여주고 지인들과 레이드 일정을 맞출 수 있게 돕는 로스트아크 원정대 운영 도구.

## 해결할 문제

다캐릭 유저는 캐릭터 조회, 성장 계산, 주간 숙제, 파티 일정 조율을 서로 다른 사이트와 메신저에서 처리한다. 이 프로젝트는 모든 로아 기능을 제공하는 포털이 아니라 다음 두 결정을 빠르게 만드는 데 집중한다.

1. 이번 주 내 원정대에서 누구에게 무엇을 먼저 투자할 것인가?
2. 원하는 사람들과 어느 레이드를 언제 함께 갈 것인가?

## 핵심 사용자

- P0: 본캐와 배럭을 여러 개 운영하는 중상위권 유저
- P1: 기존 지인, 길드원, 고정 파티와 매주 레이드 일정을 조율하는 유저
- 후순위: 복귀·초보 유저, 공개 파티를 찾는 유저, 대형 길드 운영자

## MVP 핵심 흐름

### A. 원정대 투자 판단

캐릭터 검색 → 실제 원정대 조회 → 성장 후보 비교 → 추천 1~3개와 근거 확인 → 후보 저장 → 다음 주 재확인

### B. 지인 레이드 조율

모집방 생성 → 레이드·난이도·후보 시간 설정 → 초대 링크 공유 → 참여자가 캐릭터와 가능한 시간 제출 → 방장이 최종 일정 확정

## MVP 기능

### P0. 신뢰할 수 있는 원정대 조회

- 실제 LostArk Open API 기반 원정대·캐릭터 조회
- 데이터 출처, 조회 시각, 정상/부분 성공/외부 장애 표시
- 실제 데이터와 샘플 데이터를 명확히 구분

### P0. 설명 가능한 투자 후보

- 원정대 기준 최대 3개 추천
- 대상 캐릭터, 추천 행동, 예상 비용, 기대 효과, 추천 근거, 신뢰도
- 지원 범위를 벗어난 캐릭터에는 추천을 생성하지 않음
- 추천은 정답이 아닌 의사결정 후보임을 표시

### P1. 저장과 재방문

- 원정대 및 투자 후보 저장
- MVP에서는 기기 또는 초대방 단위 저장 허용
- 영속 저장 전에는 저장 범위와 초기화 가능성을 표시

### P1. 초대형 레이드 모집방

- 방장이 레이드, 난이도, 모집 인원, 후보 시간을 설정
- 참여자는 초대 링크로 입장해 캐릭터명, 역할/포지션, 가능한 시간을 제출
- 방장은 참여 현황을 확인하고 최종 일정을 확정
- 자세한 범위는 `docs/RAID_PARTY_MVP.md` 참조

## 명시적 제외 범위

- 공개 랜덤 매칭 및 추천 알고리즘
- 실시간 채팅, 음성 채팅, Discord 대체
- 푸시 알림과 캘린더 자동 연동
- 자동 파티 조합·스펙 합격 판정
- 실제 레이드 완료 여부 자동 추적
- 실시간 거래소·시세 포털
- 랭킹, 커뮤니티, 지도·도감, 독립 계산기
- 대형 길드·공대 운영 및 권한 체계

## 성공 기준

### 투자 판단

- 테스트 유저 10명 중 7명 이상이 추천 근거를 이해한다.
- 테스트 유저 10명 중 5명 이상이 실제 투자 판단에 도움이 된다고 답한다.
- 검색 후 30초 안에 우선순위 후보 하나를 선택할 수 있다.
- 실데이터와 샘플 데이터가 혼동되는 사례가 없다.

### 레이드 조율

- 방장이 2분 안에 모집방을 만들고 링크를 공유할 수 있다.
- 초대받은 참여자가 1분 안에 캐릭터와 가능 시간을 제출할 수 있다.
- 4명 이상의 참여 응답이 있는 방 중 60% 이상에서 최종 시간이 확정된다.
- 기존 메신저만 사용할 때보다 일정 확인 메시지 왕복이 줄었다고 응답한다.
